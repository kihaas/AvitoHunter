# ТЗ — AvitoHunter

## 1. Цель проекта
Разработка автоматической системы мониторинга новых объявлений на Avito.ru по заданным параметрам товара с применением ИИ-анализа текста и изображений и мгновенной отправкой релевантных объявлений в Telegram.

## 2. Функциональные требования

### 2.1. Парсинг
- Периодический обход указанных поисковых страниц Avito.
- Обход анти-бот защиты (Playwright + ротация прокси).
- Извлечение: заголовок, цена, описание, характеристики, фото (минимум 1), ссылка, локация, наличие доставки.
- Фильтрация уже просмотренных объявлений через SQLite.

### 2.2. AI-анализ
- Анализ текста + фотографий с помощью vision-модели OpenRouter.
- Оценка по критериям:
  - Соответствие бренду и ключевым словам
  - Цена в заданном диапазоне
  - Состояние товара
  - Качество фото (дефекты, подделки)
  - Полнота описания
- Возврат структурированного JSON + итоговый score.
- Порог уведомления — `AI_MIN_SCORE` (по умолчанию 0.85).

### 2.3. Уведомления
- Мгновенная отправка в Telegram (aiogram 3.x).
- Карточка: фото + ключевые данные + ссылка.
- Поддержка личных чатов и групп.

### 2.4. Конфигурация
- Все параметры через `.env` (Pydantic Settings).
- Поддержка нескольких поисковых URL.
- Гибкие TARGET_* параметры под любой товар (ракетка, iPhone, велосипед и т.д.).

### 2.5. Надёжность
- Автоперезапуск при сетевых ошибках.
- Логирование (уровень настраивается).
- Graceful shutdown.

## 3. Нефункциональные требования
- Docker-first (одна команда запуска).
- Поддержка прокси (HTTP/SOCKS5).
- Асинхронная архитектура.
- Минимальное потребление ресурсов.
- Код должен быть модульным и легко расширяемым.

## 4. Архитектура
- **Слои**:
  - `core` — конфигурация, логи, промпты, планировщик.
  - `parser` — извлечение данных с Avito.
  - `ai` — анализ объявлений.
  - `database` — работа с SQLite.
  - `bot` — Telegram-бот (handlers + middlewares).
- **Поток**:
  1. Scheduler → Parser → DB (проверка дубликатов)
  2. → AI Analyzer → если notify=True → Bot → Telegram

## 5. Технологический стек
- Python 3.11+
- Playwright, aiogram 3, APScheduler, aiosqlite, OpenRouter, Pydantic v2
- Docker + docker-compose
