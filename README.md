# AvitoHunter 🎯

Автоматический монитор (парсер) новых объявлений на Avito с ai обработкой текстовых и медиа данных, заточенных под определенный товар, с учетом всех характеристик и отправкой уведомлений в Telegram.

## ✨ Возможности

- 🔍 **Парсинг Avito** через Playwright с обходом блокировок
- 🤖 **AI-анализ** текста объявлений и фотографий (качество, соответствие характеристикам)
- 🎯 **Умная фильтрация** по всем параметрам товара (бренд, размер, состояние, цена)
- ⚡ **Мгновенные уведомления** в Telegram с фото и описанием
- 🐳 **Docker-контейнер** — запуск за 2 команды
- 🌐 **Прокси-поддержка** (HTTP/SOCKS5) для обхода блокировок РФ
- 💾 **SQLite база** — хранение просмотренных объявлений
- 🔄 **Автоперезапуск** при ошибках сети

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/kihaas/AvitoHunter.git
cd AvitoHunter
```

### 2. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```bash
# ==================== TELEGRAM (ОБЯЗАТЕЛЬНО) ====================
TELEGRAM_BOT_TOKEN=1234567890:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
TELEGRAM_CHAT_ID=-1001234567890

# ==================== AVITO (ОБЯЗАТЕЛЬНО) ====================
# Ссылки на поиск Avito (через пробел)
AVITO_SEARCH_URLS="https://www.avito.ru/samara?q=ракетка+падел https://www.avito.ru/moskva?q=padel"

# ==================== ПРОКСИ (РЕКОМЕНДУЕТСЯ ДЛЯ РФ) ====================
HTTP_PROXY=http://user:pass@proxy-server:port
SOCKS_PROXY=socks5://user:pass@proxy-server:1080

# ==================== ПАРАМЕТРЫ ТОВАРА (ПЕРЕНАСТРОЙТЕ ПОД СВОЙ!) ====================
TARGET_PRICE_MAX=15000          # Максимальная цена (руб)
TARGET_PRICE_MIN=5000           # Минимальная цена (руб)
TARGET_BRAND="Wilson,Babolat,Head"  # Бренды через запятую
TARGET_CONDITION="новая,отличное"   # Состояние
TARGET_SIZE="3/8"               # Размер/характеристика
TARGET_KEYWORDS="карбон,графен,Pro Staff"  # Ключевые слова
AI_MIN_SCORE=0.85               # Минимальный AI-score (0.0-1.0)

# ==================== ИНТЕРВАЛ И ЛОГИРОВАНИЕ ====================
CHECK_INTERVAL_SECONDS=120      # Интервал проверки (сек)
LOG_LEVEL=INFO                  # Уровень логов (DEBUG/INFO/WARNING/ERROR)
```

> **⚠️ ВАЖНО:** Обязательно перенастройте параметры `TARGET_*` под свой товар!

### 3. Запуск через Docker Compose

```bash
docker compose up -d --build
```

### 4. Просмотр логов

```bash
docker compose logs -f avitohunter
```

## 🛠 Настройка под свой товар

### Основные параметры в `.env`:

| Параметр | Описание | Пример для теннисной ракетки | Пример для iPhone |
|----------|----------|------------------------------|-------------------|
| `TARGET_PRICE_MAX` | Максимальная цена | `15000` | `80000` |
| `TARGET_PRICE_MIN` | Минимальная цена | `5000` | `50000` |
| `TARGET_BRAND` | Бренды (через `,`) | `"Wilson,Babolat,Head"` | `"Apple"` |
| `TARGET_CONDITION` | Состояние | `"новая,отличное"` | `"новый,идеальное"` |
| `TARGET_SIZE` | Размер/параметр | `"98in2,3/8"` | `"128GB,256GB"` |
| `TARGET_KEYWORDS` | Ключевые слова | `"карбон,Pro Staff"` | `"Pro Max,A17"` |
| `AI_MIN_SCORE` | Минимальный AI-score | `0.85` | `0.90` |

### Примеры конфигураций:

#### 🎾 Теннисная ракетка:
```bash
TARGET_PRICE_MAX=12000
TARGET_BRAND="Wilson,Babolat,Head,Yonex"
TARGET_CONDITION="новая,отличное"
TARGET_SIZE="98in2,100in2"
TARGET_KEYWORDS="карбон,графен,Pro Staff,Blade"
AI_MIN_SCORE=0.85
```

#### 📱 iPhone:
```bash
TARGET_PRICE_MAX=80000
TARGET_BRAND="Apple"
TARGET_CONDITION="новый,идеальное"
TARGET_SIZE="128GB,256GB"
TARGET_KEYWORDS="iPhone 15 Pro,A17 Pro,титан"
AI_MIN_SCORE=0.90
```

#### 🚴 Велосипед:
```bash
TARGET_PRICE_MAX=45000
TARGET_BRAND="Merida,Giant,Trek"
TARGET_CONDITION="отличное,хорошее"
TARGET_SIZE="M,L"
TARGET_KEYWORDS="карбон,29 колёс,XT"

```

## 📱 Настройка Telegram

### 1. Создание бота:
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Придумайте имя и username (должен заканчиваться на `bot`)
4. Скопируйте полученный токен в `TELEGRAM_BOT_TOKEN`

### 2. Получение Chat ID:
1. Добавьте бота в чат (или начните диалог)
2. Отправьте любое сообщение боту
3. Откройте в браузере: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
4. Найдите `"chat":{"id":-1001234567890}` — это ваш `TELEGRAM_CHAT_ID`

> Для личного чата ID будет положительным, для групп/каналов — отрицательным.

## 🤖 AI-фильтрация

Бот автоматически анализирует каждое объявление по критериям:

| Критерий | Описание | Вес |
|----------|----------|-----|
| ✅ Соответствие бренду | Проверка по списку `TARGET_BRAND` | 25% |
| ✅ Цена в диапазоне | Проверка `TARGET_PRICE_MIN` – `TARGET_PRICE_MAX` | 20% |
| ✅ Состояние | Анализ текста на соответствие `TARGET_CONDITION` | 15% |
| ✅ Характеристики | Проверка размера, модели, параметров | 25% |
| ✅ Качество фото | Анализ изображений на дефекты | 10% |
| ✅ Полнота описания | Длина и информативность текста | 5% |
